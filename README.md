# -AES-SHA256-
这个项目把 AES 和 SHA256 结合起来做成一个完整的数据安全传输模 块。  其中 AES 主要负责数据机密性保护，SHA256 负责数据完整性校验。架构上我采用的 是 APB Slave + AXI4 Master ，两个模块相互独立，但采用同样的数据+控制流，可根据需要进行结合

架构上采用APB Slave + AXI4 Master的双总线解耦设计。 上游通过 APB 和模块进行交互，模块通过 AXI 接口作为 DMA 自动在总线上搬运数据。
控制流走 APB：CPU 只需要花极少的周期，把源地址、目的地址、数据长度、密钥等参数配置到我的 APB 寄存器里，然后发一个 Start 脉冲就可以去忙别的事了
数据流走 AXI Master：我内置了硬件 DMA 引擎 。由我的硬件主动去向系统内存发起读写请求，实现了加解密过程的完全脱机运行， CPU占用率极低。 

此外为了减少 AXI 读写多数据的延迟，我设计了 Outstanding 机制。在SHA 256 中，可以通过outstanding 提前去到下一个block 的 16 组 32-bit 数据，并存到input buffer 中。 而对于AES， 则是 4 组 128 bit 的缓存数据。

本项目将两个主模块进行拆分，具体每个模块的内容可以进入对应目录进行查看。
